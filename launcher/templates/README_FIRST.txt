========================================================
  Baghdo's Workshop  —  portable workspace
========================================================

HOW TO RUN
  1. Plug in this drive (or open this folder).
  2. Double-click:  Launcher.exe
  3. A black loading screen appears, then a SIGN-IN screen.
  4. Create a profile (a name; a PIN is optional), then sign in.
  5. The DESKTOP opens — click a tool to use it. Sign out from the top-right.

WHAT'S INSIDE
  - Visual Builder (drag-and-drop), Notes, Labels, Recipe Card,
    Cross-Stitch, Pricing — plus any apps you drop in the apps folder.
  - Each profile has its own private projects and data.

ADD MORE APPS
  - Drop any self-contained .html file into the  apps  folder.
    It will show up on the desktop automatically.

WHERE THINGS GO (all inside this folder)
  Launcher.exe   the program (double-click this)
  apps/          drop extra .html apps here
  exports/       finished files you export
  logs/          startup.log, app.log
  app-data/      profiles + settings (your private data)
  resources/     bundled assets

NOTES
  - No Node.js, no install, no internet needed. Fully offline.
  - Uses Microsoft WebView2, built into Windows 10 & 11.
  - The PIN is a simple local lock for convenience, not strong encryption.
